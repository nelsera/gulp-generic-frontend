O que é este documento?

A Guideline contém todos os padrões de desenvolvimento front-end do projeto.
Todos os módulos e páginas "mock-up" do projeto estão disponíveis neste site.

Todos os estilos definidos pela Guideline não impactam em regras globais de CSS. Em todos os casos as regras são aplicadas através de classes explícitas, como o caso da .gl com comportamento diferente de acordo com a tag que é aplicada. Algumas variações da classe é usada como prefixo: .gl-header, .gl-char-table, etc.

O site é organizado em algumas categorias como Tipografia, Cores, Grids e Módulos:

Tipografia#

Todas as famílias tipográficas estão disponíveis aqui, assim como pesos, tamanhos de fontes usados no projeto, ícones e regras globais de diagramação de textos.

Cores#

Todas as cores do projeto são lidas do arquivo de cores e listadas de forma que seja possível visualizar qualquer inconsistência.

Há também um gerador de cores que permite comparar se a nova cor a ser criada é muito parecida com alguma que já existe.

Grids#

Todas as grids criadas para o projeto estão listadas aqui, não só para versões responsivas, mas para situações onde uma nova grade é necessária para suprir necessidades específicas de um layout.

Módulos#

Todos os módulos do site e regras base de espaçamento, z-index. Estão organizados por categorias e devem seguem o conceito de atomic design, como disposto na apresentação sobre o Hyojun.sass-standards.
